# Security Policy

## Supported versions

Textream is pre-1.0. Security fixes target `main` until tagged releases exist.

## Reporting a vulnerability

Please do not open a public issue for a vulnerability.

Email the maintainer or use GitHub private vulnerability reporting if enabled. Include:

- affected version or commit
- macOS version
- reproduction steps
- impact
- suggested fix, if known

## Privacy notes

Textream uses the microphone only for local voice activity detection. It does not record, persist, transcribe, upload, or share audio.
